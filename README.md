# 🧮 Age Calculator  

A **modern, mobile-first Age Calculator** built with **HTML, CSS, and JavaScript**.  
Includes smooth animations, a sliding modal for error handling, and celebratory confetti when it’s your birthday 🎉.  

---

## ✨ Features
- 📱 **Mobile-first responsive design** with a clean, modern UI  
- 🗓️ Enter your **day, month, and year of birth**  
- 🚫 Error handling with a **sliding modal** if inputs are invalid or incomplete  
- 🎂 **Birthday detection**:
  - Shows a **Happy Birthday card + confetti animation** on your birthday  
  - If not your birthday, shows the **next birthday date** and **days left** until it  
- 📊 Displays detailed results:
  - **Age**  
  - **Birth date**  
  - **Next birthday + days left**  
  - **Life stage** (Infant, Toddler, Teen, Adult, etc.)  
  - **Generation** (Silent, Boomers, Gen X, Millennials, Gen Z, Alpha)  
  - **Zodiac sign**  
  - **Zodiac color** (row styled with corresponding color 🎨)  
  - **Zodiac characteristics**
